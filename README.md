# AstralPalm – Palm Reading App

A premium AI-powered palm reading iOS app focused on self-discovery, wellness, and mystical insight.

---

## Product Overview

AstralPalm generates AI-powered palm readings from user-uploaded hand photos. The MVP focuses on a simple conversion funnel: upload palms → preview → unlock full report. The system supports anonymous users and optional signup. Free users receive a preview reading, while paid users unlock a full report generated by AI.

**Primary goals:**
- Deliver a fast and engaging palm-reading experience
- Minimize friction before payment (no forced signup)
- Encourage signup after purchase for data persistence
- Store user data on server with temporary retention for free users and permanent retention for paid users

## User Flow

```
App Launch → Loading → Intro → Home → Palm Reading → Results → Repeat Engagement
```

### Core User States

| State | Description | Flow |
|---|---|---|
| New User | No palm uploaded | Upload → Quality Check → Scan Animation → Preview → Paywall → Optional Signup → Analysis → Full Report |
| Uploaded, Not Paid | Preview exists, report locked | Preview → Unlock Button → Paywall |
| Paid User | Full report unlocked | Full Report (revisit anytime) |

### Home Screen Dynamic CTA

| State | Button Text |
|---|---|
| No reading exists | Start Palm Reading |
| Preview exists, not paid | Continue Reading |
| Full report unlocked | View My Report |

## Screen Descriptions

### Loading Screen
- App initialization, load config and user session
- Duration: 1–2 seconds max
- AstralPalm logo with subtle cosmic animation background

### Intro Screen
- Shown only on first launch
- Value statement: *"Discover what your palms reveal about your destiny."*
- CTA: **Get Started**

### Home Screen
- Main navigation hub: Palm Reading + Horoscope (Coming Soon)
- Persistent top-right signup/login entry point
- Anonymous users see: Sign In / Sign Up
- Signed-in users see: Profile icon

### Hand Photo Upload
- Upload left and right palm photos
- Visual guide for correct palm positioning
- Requirements: good lighting, entire palm visible, fingers slightly spread

### Photo Quality Check
- Validates: palm detected, brightness sufficient, full palm visible, not blurry
- On failure: prompts user to retake photo before continuing

### Short Scan Animation (Preview Generation)
- Duration: 1–1.5 seconds
- Messages: *Scanning palm lines… Detecting major lines… Preparing your preview*
- Transitions into Preview screen

### Preview Screen
- 2–3 short teaser insights from the palm
- Feels intriguing but incomplete
- Primary CTA: **Unlock Full Reading**

### Paywall
- Benefits of full report: life line analysis, love & relationships, career potential, left vs right palm comparison
- Price and purchase button
- CTA: **Unlock My Full Reading**

### Optional Signup (Post-Payment)
- Prompt to create account for cross-device access
- Message: *"Create an account to save your reading and access it anytime."*
- Options: Sign Up / Maybe Later — fully optional

### Long Analysis Animation (Full Report Generation)
- Duration: 4–6 seconds
- Sequence: *Analyzing Life Line → Interpreting Heart Line → Comparing both palms → Mapping your destiny pattern*
- Cosmic/star animation matching branding

### Full Report Screen
Sections:
- Personality Insights
- Life Path Analysis
- Love and Relationships
- Career Potential
- Destiny Overview

Paid users can revisit anytime.

## Subscription & Monetization

- **Model:** One-time purchase to unlock full report (per reading)
- **Conversion strategy:** Preview hooks user before paywall; signup encouraged post-payment
- **No forced account** — anonymous usage supported via `anonymous_user_id`

---

## Data & Privacy

- Anonymous `user_id` generated locally on first use, sent with all API requests
- If user signs up, anonymous data is linked to their account
- **Free users:** server data retained temporarily (48 hours)
- **Paid users:** data stored permanently on server
- Users can delete their scans
- Clear user consent obtained before any upload

### Data Storage

| Location | Data |
|---|---|
| Local | `anonymous_user_id`, `last_reading_id` |
| Server | Palm images, analysis results, preview text, full report, payment records |
